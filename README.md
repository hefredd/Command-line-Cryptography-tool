# Cryptography Tool
A Python-based cryptography and password security toolkit designed for cybersecurity learning and secure data handling.
## This toolkit allows users to:
-Generate SHA hashes for files
- Verify file integrity
- Encrypt and decrypt messages using AES
- Encrypt and decrypt messages using RSA
- Check password strength
- Securely hash and verify passwords
# Features
## 1. File Hashing
Generate secure SHA hashes for files to detect tampering or unauthorized modifications.

## 2. File Integrity Verification
Compare hashes between files to determine whether files have been altered.

## 3. AES Encryption & Decryption
Encrypt and decrypt messages using AES symmetric encryption.

## 4. RSA Encryption & Decryption
Secure messages using RSA public/private key cryptography.

## 5. Password Security
- Password strength checker
- Secure password hashing
- Password verification

# Technologies Used
- Python 3
- Cryptography
- Hashlib
- Getpass
- RSA/AES Encryption
